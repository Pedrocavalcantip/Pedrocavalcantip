Projeto em Destaque: CinboraImpactar

Descrição: O CinboraDoar é uma plataforma web desenvolvida para o projeto Bora Impactar da Prefeitura, com o objetivo de otimizar a gestão de doações entre ONGs. O sistema permite que ONGs solicitem itens de necessidade e também repassem itens excedentes para outras instituições. O grande desafio foi criar uma interface extremamente simples e intuitiva, focada em usuários idosos com pouca afinidade tecnológica.

Minhas Contribuições: Atuei como Scrum Master da equipe de front-end, organizando sprints e garantindo a entrega contínua de valor. Participei ativamente do desenvolvimento da interface com React.js, focando em usabilidade e acessibilidade. Na fase de integração, tive um papel crucial no back-end, desenvolvendo e implementando testes de ponta a ponta para garantir a segurança da API e a integridade dos dados.
